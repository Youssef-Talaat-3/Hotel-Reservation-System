# Hotel-Reservation-System
Web-based Hotel Reservation System using HTML, CSS, JavaScript

## Overview
The Hotel Reservation System is a web-based application designed to allow users to search for hotels, view detailed information, and make room reservations online. The system aims to simplify the hotel booking process through a clean and user-friendly interface.

---

## Objectives
- Provide an easy-to-use platform for hotel searching and booking
- Allow users to view hotel details before booking
- Manage booking information efficiently using client-side storage
- Apply concepts of System Analysis and Design in a practical project

---

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Browser LocalStorage

---

## System Features
- User registration and login
- Hotel search functionality
- Display list of available hotels
- View hotel details and room pricing
- Booking and checkout process
- Booking confirmation page
- Client-side data handling using LocalStorage

## How to Run the Project
1. Download or clone the repository
2. Open the `frontend` folder
3. Run `index.html` in any modern web browser
4. No server or backend setup is required
